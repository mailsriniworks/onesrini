## How to Use

### Viewing the Website
Simply open `index.html` in any modern web browser. The website is a static HTML page with no build process required.

### Local Development
1. Clone the repository:
   ```bash
   git clone https://github.com/mailsriniworks/onesrini.git
   cd onesrini
   ```

2. Open `index.html` in your browser:
   ```bash
   # On Linux/Mac
   open index.html
   
   # Or use a local server (recommended)
   python -m http.server 8000
   # Then visit http://localhost:8000
   ```

### Customization
To customize this resume for your own use:

1. **Personal Information**: Edit `index.html` sections:
   - Update name, contact details in the About section
   - Modify LinkedIn badge URL
   - Update social media links

2. **Content**: Update the following sections in `index.html`:
   - Experience entries
   - Education history
   - Skills and tools
   - Interests description
   - Awards and certifications

3. **Profile Picture**: Replace `assets/img/profile.jpg` with your own photo

4. **Styling**: Modify `css/styles.css` to change:
   - Color scheme (primary color)
   - Fonts
   - Layout spacing

## Deployment

This is a static website that can be deployed to any web hosting service:

### GitHub Pages
1. Go to repository Settings → Pages
2. Select branch (e.g., `main`) as source
3. Website will be available at: `https://mailsriniworks.github.io/onesrini/`

### Other Options
- **Netlify**: Drag and drop the folder or connect to GitHub
- **Vercel**: Import the GitHub repository
- **AWS S3**: Upload files to an S3 bucket with static hosting
- **Traditional hosting**: Upload via FTP/SFTP
